Leaflet Tutorial

Introduction

What is Leaflet.js?

Leaflet is a light weight open sources java script library that allows you to quickly creat comprehensive and interactive maps. It takes all the trouble out of trying to intergrate and create that nice or custom map within your application. Leaflet is versitile and developed for more moblie friendly interactive maps. It weighs only about 38kb of javascript and has many robust features.

Leaflet is the leading open-source JS library for interactive mobile focused maps. It is an easy to use library with extensive documentation and can be extened with a number of plugins and add ons. Leaflet takes the pain out of making custom maps and markers, and streamlines many complex actions/code into simple and easy to implement functions/actions. Leaflet is highly compatible with most, if not all modern browsers and works well with a number of different devices.

Summary:

  • Easy to understand and use
  • Customizable
  • Lightweigtht
Setup
  • Accounts

    In order to utilize Leaflet.js you will have to make an account with a web services provider like Google, MapBox, or MapQuest. Registering an account will provide you with an API key, appid, and documentation that you will need to create and style your own maps. It is a simple process and most are free or have a pay threshold, so many requests per day/month/week etc. Once you have made an account you will have to go into your dashboard and aquire your api key. Be mindful that some providers will have you register you application and give it an id along with utilizing a api key.

  • API and appid

    The api key is a unique identifyer that allows your application to access the Map api's resources. The key, or token depending on the provider, is the only way that you can use said service. Most API's are restful which means that you will be utilizing a series of urls to access different aspects of a providers web service. For this example once you have an api key/token all you need to do is place it in your map attributes in your code and that is it.Note: Some providers will have you create or register your individual app and give it an id.

  • IDE

    First you are going to want to pick an IDE to develop the web application in. An IDE is an Intergrated Development Environment. For this demo I have picked Visuial Studio 2017: Get VS Here! All you need to do is make a new Project. Then go to the sub section web and pick web application. Next you pick an empty form and VS should generate the project. Then you want to right click add item on the project icon. When the dialog box appears you choose html file. Now do this to add a CSS and JS file if you want.

  • Software:

    You can use leaflet one of two ways. Either through down loading the package into your IDE or through a CDN. If you use a CDN all you need to do is include these lines of code within the header of your html page. The first line is the Leaflet CSS:

Simple Example 1: layers and markers

For this first example we will get a tile set and place a marker on the map.

After you have added the CSS and JS files from above into your header section on your html page. You need to place a div or section element into the main body of your page,like so:

< div id="map1">< /div>

Then you will want to give it some styling on your CSS page. I just gave it a height in pixels.

#map1{height: 300px;}

Next you will have to set the location and and zoom level of the map. To do this you have to pass setViews a set of coordinates and a integer zoom level and place that into a variable.

var myMap = L.map('map1').setView([39.9526, -75.1652], 13);

Then you will have to add a tile layer to your map. In order to do this you will have to pick a map API like MapBox or Google Maps API like the ones in the Resources section.

Now you will have to add the tile layer using L.tileLayer();. You will need the URL template, an attribution text,app id, and the max zoom level for that layer. For MapBox the code looks like this:

L.tileLayer('https://api.tiles.mapbox.com/v4/{id}/{z}/{x}/{y}.png?access_token={accessToken}', {
attribution: 'Map data © < a href="https://www.openstreetmap.org/" >OpenStreetMap contributors, < a href="https://creativecommons.org/licenses/by-sa/2.0/">CC-BY-SA, Imagery © < a href="https://www.mapbox.com/">Mapbox',
maxZoom: 18,
id: 'mapbox.streets',
accessToken: 'access code/ API key goes here'
}).addTo(myMap);

The id is the type of map of tile set you may want to use. Here is a list for MapBox: Map Styles. The map id will generally be one of these but in the format of mapbox.styleyouwanttouse. For example if I wanted to use the satellite tile set I would put mapbox.satellite in the section of the id.

The last line of code .addTo(myMap); adds the tile set to the map variable from earlier. It should look something like this:

Finally you will want to add a marker to the map. To do this you will have to pick a set of coordinates on the map, lat and long, as to where you want to place the marker. First make a var and then have it equal to the L.marker(). Once again you will have to add the marker to the tile set using thefucntion with the map variable .addTo(yourmapvariable);. It should look something like this:

var firstMarker = L.marker([39.952515, -75.163544]).addTo(myMap);

Simple Example 2 custom markers with popups

Building off the previous example I will now show you how to add a popup to any marker or element that you place on the map. It is fairly easy process due to Leaflet.js. All you need to do is take the var firstMarker from the previous example and utilize the .bindPopup("Your message here") with the .openPopup(); method. It will look something like this:

firstMarker.bindPopup("My very own marker with a popup").openPopup();

Example 3 Custom images and shapes

There are different classes to create various objects like circles, rectangles,lines and even how to use and customize your very own markers. Here is an example of using the rectangle class.

var bounds = [[some map latitude, some map longitude], [another map latitude, another map longitude]]; L.rectangle(bounds, {color: "enter hex color here", weight: numerical value like 1}).addTo(yourMap); yourMap.fitBounds(bounds);

The var bounds just needs two points on the map with the lat and long. Then you use the L.rectangle class to add the bounds to the shape, add a hex color and then the numerical of the shape. Finally you use the map map variable you have been using and force the map to zoom in on the shape.

Summary

Leaflet.js is a javascript framework library that allows you to quickly and easily build fun and interactive maps. It uses any Map API and has very easy to follow documentation. It takes the time and frustration from making your own mobile friendly maps into a bearable and down right pleasent experience. There is even more to the framework that I did not get a chance to touch such as Geocoding and GeoJSON but that could be for a later time/tutorial.
